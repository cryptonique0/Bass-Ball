================================================================================
                              HOW IT WORKS
================================================================================

DURING MATCH (0-90 minutes):
  1. Player performs action (shoot, pass, tackle, sprint)
  2. Match engine updates gameState
  3. Statistics are recorded in matchStats
  4. Event is added to events log
  5. UI displays live stats and event log

AT 90 MINUTES:
  1. Match engine reaches gameTime >= 90
  2. LiveMatch detects end of match
  3. Component switches from match controls to MatchResults
  4. MatchResults component mounts

RESULTS DISPLAY:
  1. MatchResults parses events log
  2. Extracts player statistics
  3. Calculates MVP (by goals ‚Üí assists ‚Üí impact)
  4. Renders beautiful summary screen
  5. User can restart via button

GOAL SCORING WITH ASSISTS:
  1. Player shoots successfully
  2. scoreGoal() called
  3. findLastPasser() searches recent passes
  4. Assist maker found and credited
  5. Event recorded with assist information
  6. Both stats updated (goal + assist)
